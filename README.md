YUI Flickr Carousel
===================

Dependencies
------------
*   scrollview
*   scrollview-paginator
*   base-build
*   get

Instantiation
-------------
```javascript
new Y.FlickrCarousel({
    srcNode: Y.one('.main-carousel'),
    apiKey: 'XXXXX',
    photosetId: 'XXXXX',
    autoAdvance: true,
    startDelay: 3000,
    advanceDelay: 3000,
    width: '500px'
});
```
